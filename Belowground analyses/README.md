## Folder and code content
In this folder are provided the datasets used in [publication].
Analyses of the belowground nutrient content are described in the code. Specifically, are present:
- Linear Mixed Models (LMMs) for nitrogen (N), available phosphorus (AP), and soil organic carbon (SOC) concentrations in the topsoil
- Generalised Least Square Models (GLS) for SOC content in the topsoil
- Plots creation for SOC content
