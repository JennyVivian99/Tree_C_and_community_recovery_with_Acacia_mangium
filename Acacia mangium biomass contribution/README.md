## Folder and code content
In this folder, datasets used in [publication] are provided, along with the code containing the analyses related to the biomass contribution of _Acacia mangium_ and other tree species.
In addition, the dataset 'Biomass_Species_Contribution' is available for a handy visualisation of the results. This dataset is not needed or used in the analyses.
The code contains:

- Linear Mixed Models (LMMs)
- Generalised Least Square models (GLS)
- Plots creation
